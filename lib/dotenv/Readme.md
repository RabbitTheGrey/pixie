# Dotenv
### Парсер переменных окружения из файла .env ###

Отличительной особенностью данного парсера является кеширование переменных окружения без постоянного сканирования файла или перемещения переменных в OS.

### Использование ###
```go
// Пример получения переменной в коде программы:
import "pixie/lib/dotenv"

func main() {
    token := dotenv.Getenv("JWT_TOKEN")
}
```
